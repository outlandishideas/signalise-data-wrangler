# Actual appt start time

For all Signalise appointments with CCGs this
is the booking time.

## Monday

This is the same as the `start__hour` from 
`monday.booking_sales_pipeline_2020_21`.

## Reporting

The `actual_start_time` is set as the booking
start time.

## Worker

as `HH:MM`
